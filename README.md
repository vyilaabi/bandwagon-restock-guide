## 搬瓦工是什么，为什么总缺货

搬瓦工是加拿大 IT7 Networks 旗下的 VPS 品牌，2004 年成立，专注于高性价比的 KVM 架构服务器。面向中国大陆用户，它最大的卖点是接入了 CN2 GIA、CMIN2 等优质直连线路，晚高峰速度稳定，延迟比普通国际线路低得多。

**搬瓦工补货（restocking）**，指的是搬瓦工限量版套餐售罄后重新上架的行为。由于限量版套餐数量有限、价格低、性价比高，每次补货后往往几小时到一两天内再次售罄。常规套餐则长期有货，随时可以购买。

这是关键区分：**常规套餐随时能买，限量版套餐需要等补货。**
